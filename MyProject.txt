
1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

Для того, чтобы скопировать репозиторий с GitHub на свой компьютер, вам необходимо выполнить следующие действия:

Откройте страницу репозитория на GitHub.com.
Нажмите на кнопку {Code} и скопируйте URL-адрес репозитория.
Откройте терминал на своем компьютере и перейдите в папку, в которую вы хотите скопировать репозиторий.
Введите команду git clone и вставьте скопированный URL-адрес репозитория.

Как получить на локальный компьютер изменения с github.com?
Чтобы получить изменения с GitHub на локальный компьютер, вам необходимо выполнить следующие действия:

Откройте терминал на своем компьютере и перейдите в папку, в которой расположен локальный репозиторий.
Введите команду git pull, чтобы получить изменения с GitHub.

Какую информацию нам дают команда :
git status?
Команда git status показывает текущее состояние вашего рабочего дерева и индекса. Она сообщает о том, какие файлы были изменены, добавлены или удалены.

git log?
 Команда git log показывает историю коммитов для текущей ветки. Она сообщает о том, кто сделал коммит, когда он был сделан, и какое сообщение было добавлено к коммиту
